# The Odin Project - Recipes

## About

A recipe website project from [The Odin Project](https://www.theodinproject.com/lessons/foundations-recipes).

## Insights

The text editor I use, VS Code, shows a warning highlighting the used character with the unicode hex value of U+0131, "ı", indicating that the character could be confused with the character U+0069, "i". I ignored the warning since I did not use the mentioned character in any code identifier, file name, etc. but **within the content** of the webpages. So their resemblance cannot pose any practical issue.
